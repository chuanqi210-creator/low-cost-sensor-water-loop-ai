# outputs 目录说明

这里保存所有模拟输出、训练记录和 agent 报告。

重点目录：

- `full_chain/`：全链条报告。
- `scenario_sweep/`：多场景扫查。
- `closed_loop_robustness/`：鲁棒性结果。
- `design_sensitivity/`：低成本传感设计敏感性。
- `agent1_*` 至 `agent34_*`：各 agent 报告。
- `soft_sensor_training/`：训练数据、训练指标和训练报告。

后续新增 agent 报告必须放在 `outputs/agentXX_agent_name/`。
